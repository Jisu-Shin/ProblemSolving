### <img src ="https://d2gd6pc034wcta.cloudfront.net/tier/12.svg" width="25"> 중량제한 (22-01-09 성공)

조건을 조금만 변경하였는데 바로 성공하였다

**다리무게가 중량보다 클경우만 이동이 가능하다** 이게 포인트다
